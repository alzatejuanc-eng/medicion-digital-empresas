## Configurar Google Sheets (una sola vez)

1. Crear Google Sheet en [sheets.google.com](https://sheets.google.com)
2. **Extensiones → Apps Script** → pegar el código del botón ⚙️ en la app
3. **Implementar → Aplicación web → Cualquier persona → Copiar URL**
4. En `index.html` pegar la URL:
```javascript
const CFG = {
  SHEET_URL: 'https://script.google.com/macros/s/XXXX/exec',
  ORG: 'Transformación ExO',
  VERSION: 'MIEDE 2.0 · 2025–2026'
};
```
5. Commit → GitHub → Vercel redespliega solo

**Hojas generadas automáticamente:**
- `Evaluaciones` — una fila por empresa con scores de los 8 pilares
- `Criterios Detalle` — una fila por cada uno de los 40 criterios respondidos
